# CHS Tech News

Micro-Blogging site for Charleston Tech Community

## Requirements

* NodeJS [http://nodejs.org](http://nodejs.org)
* Grunt `npm install grunt-cli -g`
* Bower `npm install bower -g`

## Setup

``` sh
cd [project]
npm install
bower install
grunt serve
```

## Run Dev Server

``` sh
grunt serve
```
